# Niskapi Co. – Coffee Shop Menu Website

## Project Documentation

### 1. Website Concept
Niskapi Co. is a coffee shop menu website created to display the different types of coffee available in the shop. The website allows users to browse the menu digitally instead of using printed menus. A simple login feature is included to simulate user access before interacting with certain menu options. The design focuses on clarity, ease of use, and responsiveness across devices.

### 2. Features and Functionality
- Coffee Menu Display: Shows available coffee products with names, images, and prices.
- Login System (Demo): Users are required to log in before proceeding with menu interactions.
- Responsive Design: The website adjusts properly on mobile phones and desktop screens.
- Navigation Menu: Includes a standard navigation bar and a hamburger menu for mobile view.
- Interactive Elements: Buttons, menu items, and pop-up messages respond to user actions.
- Basic Accessibility Support: Keyboard navigation and clear UI feedback are provided.

### 3. Technologies Used
- HTML5: Used to structure the website content and layout.
- CSS3: Used for styling, layout design, and responsiveness.
- JavaScript: Used to handle user interaction, login behavior, and pop-up messages.
- Local Storage: Used to store login status for demonstration purposes.

### 4. Conclusion
The Niskapi Co. website demonstrates basic frontend web development skills by combining structure, design, and interactivity. It provides a simple and effective way to present a coffee shop menu while incorporating a simulated login system for user interaction.
